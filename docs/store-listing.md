## Full Description

### PromptVault — Your AI Prompt Library

Stop re-typing the same prompts. PromptVault lets you save your best AI prompts and inject them into any AI chat with a single keyboard shortcut.

**How it works:**
1. Save your frequently-used prompts organized by spaces and tags
2. Press ⌘K (or Ctrl+K) on any supported AI chat
3. Search, select, and inject — done in under 2 seconds

**Supported AI tools:**
- ChatGPT (chat.openai.com)
- Claude (claude.ai)
- v0 by Vercel (v0.dev)

**Features:**
- Fast client-side search across all your prompts
- Tag-based organization with filterable tabs
- One-click copy or inject into active chat
- Cloud sync across devices
- Offline access from local cache
- Dark mode support
- Keyboard-first navigation (arrow keys + Enter)

**Pro Features (€8.99/mo):**
- Unlimited prompts (Free: 50)
- Version history with diff viewer
- Team sharing with roles
- Usage analytics
- Bulk import/export (CSV, JSON)

**Privacy:**
- No tracking cookies
- Data stored in EU (GDPR compliant)
- We never read your AI conversations
- Open source roadmap
